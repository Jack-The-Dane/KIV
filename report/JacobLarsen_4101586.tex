\begin{abstract}
%The abstract should provide a comprehensive overview of your task, detailing the problem you are addressing, the algorithm you have selected/developed/applied, the dataset you used for evaluation, and the results you obtained. This paper presents the implementation and training of a convolutional neural network (CNN) in combination with a long short term memory (LSTM) model, for detection of active seizures in electroencephalogram (EEG) signals.
The model operates on minimally preprocessed data, which enables easy and, more importantly, fast preprocessing, so that the model could potentially be used in a real-time context. The model hyper-parameters were tuned via a grid-search algorithm, leading to a highly performant model. 
The resulting model was compared to a simpler machine learning model, both on the same dataset, but also on a more complex dataset to show that the more advanced model is more generalisable.
